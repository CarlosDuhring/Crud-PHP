# CRUD PHP

##  Sobre o Projeto

Este projeto é um sistema CRUD (Create, Read, Update, Delete) desenvolvido em PHP, com o objetivo de gerenciar dados de forma simples e eficiente. A aplicação ainda está em desenvolvimento e faz parte do aprendizado e prática de desenvolvimento web com integração a banco de dados.

##  Status do Projeto

 Em desenvolvimento

##  Funcionalidades (atuais e planejadas)

* Cadastro de dados
* Listagem de registros
* Edição de informações
* Exclusão de registros
* Integração com banco de dados
* Interface web simples

##  Tecnologias Utilizadas

* **Backend:** PHP
* **Frontend:** HTML, CSS
* **Banco de Dados:** MySQL

## Estrutura do Projeto

```
Crud-PHP/
│-- index.php
│-- conexao.php
│-- cadastrar.php
│-- editar.php
│-- excluir.php
│-- style.css
```

## Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/CarlosDuhring/Crud-PHP.git
```

2. Coloque o projeto em um servidor local (XAMPP, WAMP ou similar).

3. Configure o banco de dados MySQL.

4. Ajuste as credenciais no arquivo de conexão (`conexao.php`).

5. Inicie o servidor Apache e MySQL.

6. Acesse no navegador:

```
http://localhost/Crud-PHP
```
